## 📦 ビルド方法

```
# 依存関係パッケージをインストールします
$ yarn install

# ホットリロードを実行する: localhost:3000
$ yarn dev

# 本番環境とローカルサーバーの起動
$ yarn build
$ yarn start

# 制的プロジェクトを生成する
$ yarn generate
```
